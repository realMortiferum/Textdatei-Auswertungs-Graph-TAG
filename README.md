# Textdatei-Auswertungs-Graph TAG

<p>Auslesen von Integer und String-Werten aus einer Datei (konfiguriert für Result Tree von JMeter)</p>
<p>Erstellen eines Pie- und eines Bar-Charts um die Relation zwischen den Daten sichtbar zu machen</p>
<p>Kann beliebig für alle Textdateien konfiguriert werden, deren Inhalt visualisiert werden soll</p>
